## مولد أكواد QR مع شعار (PHP + Bootstrap 5)

تطبيق ويب بسيط يعمل على PHP يعرض واجهة Bootstrap 5 لتوليد أكواد QR غنية بالخصائص مع إمكانية وضع شعار في المنتصف، ودعم كامل للروابط، الاتصالات الهاتفية، الرسائل القصيرة، رسائل واتس أب، البريد الإلكتروني، الإحداثيات، وبطاقات الاتصال (vCard) المتوافقة مع أندرويد و iOS و ويندوز.

### المتطلبات
- PHP 8.1 أو أحدث.
- اتصال بالإنترنت لتحميل مكتبة واجهة المستخدم من CDN (Bootstrap و QR Code Styling).

### التشغيل محليًا
1. ثبّت الاعتمادية (لا توجد حزم Composer مطلوبة في هذه النسخة).
2. من جذر المشروع شغّل خادم PHP المدمج:
   ```bash
   php -S localhost:8000 -t public
   ```
3. افتح المتصفح على: http://localhost:8000

### أهم الخصائص
- أنواع محتوى متعددة: روابط، نص حر، اتصال هاتفي، SMS، واتس أب، بريد إلكتروني، موقع جغرافي، vCard.
- توليد تلقائي للروابط (tel:, sms:, mailto:, wa.me, geo:) مع ترميز الحقول الحساسة.
- شعار في منتصف الكود مع شعار افتراضي وإمكانية رفع شعار مخصص (PNG أو SVG فقط).
- تحكم في المقاس، الحافة الصامتة (margin)، الألوان، شكل النقاط والزوايا، ومستوى تصحيح الأخطاء.
- إمكانية تفعيل/تعطيل الشعار مع خيار إعادة الشعار الافتراضي بسهولة.
- تنزيل بصيغة PNG (افتراضي) أو SVG مع معاينة مباشرة.
- زر لاكتشاف الموقع الحالي عبر المتصفح وزر لنسخ البيانات التي تم توليدها.
- واجهة ثنائية اللغة (عربي / إنجليزي) مدمجة في النصوص والعناوين.

### ملاحظات
- يتم الاعتماد على مكتبة **qr-code-styling** عبر CDN لتوليد ومعاينة الأكواد مباشرة في المتصفح، لذلك لا يحتاج الخادم إلى مكتبات إضافية.
- في حالة العمل دون اتصال، يمكنك استبدال روابط الـ CDN بنسخ محلية من الملفات لاحقًا.
